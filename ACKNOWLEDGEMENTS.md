# Acknowledgements

Curator for Zotero began as an independently maintained continuation of [`kujenga/zotero-mcp`](https://github.com/kujenga/zotero-mcp) by Aaron Taylor. The original project established the compact FastMCP/Pyzotero foundation for Zotero search, metadata lookup, and indexed full-text access.

This repository keeps the MIT license and preserves the upstream copyright notice while adding a research-workflow oriented package, CLI, client setup helpers, diagnostics, central configuration, safer write controls, and a path toward easier distribution.
